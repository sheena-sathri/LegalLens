I need you to rebuild the LegalLens frontend as a React + Tailwind CSS application. Keep the existing FastAPI backend completely untouched — it works and serves the API on port 8000. Build a modern React frontend that talks to the backend API.

## TECH STACK FOR FRONTEND:
- React 18 with Vite
- Tailwind CSS for styling
- React Router for navigation
- Lucide React for icons
- Recharts for dashboard charts
- Axios for API calls
- The frontend should be served alongside FastAPI — either through FastAPI serving the built static files, or running on a separate port with proxy

## DESIGN SYSTEM:
- Clean, minimal, professional SaaS aesthetic — inspired by Linear, Notion, Checkbox.ai
- Dark sidebar (slate-900) with light content area (gray-50/white)
- Primary color: Indigo-600 (#4F46E5)
- Accent: Emerald for success, Amber for warning, Rose for critical
- Font: Inter (import from Google Fonts)
- Rounded corners (rounded-lg), subtle shadows (shadow-sm), clean borders (border-gray-200)
- Consistent spacing: p-6 for page content, gap-4 for grids
- Cards with white background, border, subtle shadow
- Status badges: small rounded pills with color coding
- Transitions on hover for interactive elements
- NO harsh colors, NO gradients, NO overly decorative elements
